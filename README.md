# Star Wars Landing Page

## Descripción

Landing page inspired by the Star Wars universe, built with HTML and CSS to practice visual composition and themed UI design.

## Tecnologías

- HTML
- CSS

## Objetivo

Crear una landing temática para reforzar composición, estilos y estructura frontend.

## Funcionalidades

- Diseño temático
- Secciones visuales
- Estilos CSS personalizados
- Maquetación estática

## Aprendizajes

Práctica de creatividad visual, layout y presentación de una página temática.

## Mejoras futuras

Añadir responsive, transiciones y enlace de despliegue.
